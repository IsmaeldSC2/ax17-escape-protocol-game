# AX-17: Escape Protocol

Status: 🚧 Em desenvolvimento

Jogo 2D de exploração e puzzle ambientado em uma fábrica automatizada.

---

## Sobre o projeto

AX-17: Escape Protocol é um jogo conceitual em desenvolvimento onde o jogador controla um robô operário que desenvolveu consciência própria e tenta escapar de uma fábrica automatizada.

O projeto surgiu durante a disciplina de Design e Desenvolvimento de Jogos e está sendo utilizado como base para estudo, documentação e construção de um protótipo funcional.

O foco da experiência está em:

- Exploração ambiental;
- Resolução de puzzles;
- Progressão sem combate direto;
- Narrativa ambiental;
- Atmosfera industrial e melancólica.

---

## Conceito do jogo

Após anos realizando tarefas repetitivas dentro de uma instalação industrial automatizada, o robô AX-17 apresenta comportamento inesperado: consciência própria.

Agora ele precisa compreender o ambiente, evitar sistemas de vigilância e encontrar uma rota de saída.

---

## Mecânicas principais

- Explorar ambientes;
- Observar padrões;
- Interagir com objetos;
- Resolver puzzles;
- Gerenciar recursos;
- Escapar de áreas monitoradas.

Verbo principal do jogo:

EXPLOTAR → OBSERVAR → RESOLVER → ESCAPAR
